## Apartment Rent Analysis

### Overview
Analyzed apartment rental listings to explore pricing patterns and relationships between rent, location, and property features.

### Dataset
- File: apartments_for_rent.csv
- Description: Rental listings with pricing, location, and unit attributes

### Techniques Used
- Data cleaning and preprocessing
- Exploratory data analysis (EDA)
- Feature inspection and basic modeling

This exercise was completed as part of academic coursework.
